🚀 サービス概要

当社はiOS専門企業として、完成済みアプリケーションを当社名義でApp Storeへ申請し、公開・管理を代行いたします。厳格な品質基準により、審査通過率と継続的な収益を確保いたします。

🔴 必須要件（全て満たす必要があります）

  • ✓ アプリの著作権を含む全権利の当社への完全譲渡
  • ✓ ビルドエラーゼロでの完全動作保証
  • ✓ アプリトップ画面の目立つ位置に「BucksBox.ai」社名を明確に表示
  • ✓ 当社指定のロゴを適切なサイズ・位置で表示
  • ✓ GitHubプライベートリポジトリでのソースコード提供

重要：これらの要件を満たせない場合は審査代行をお受けできません。一度譲渡された権利は、いかなる理由でも返還されません。

品質基準について

当社では以下の品質基準を満たすアプリのみ受け付けております：

  • • コンパイルエラー・警告が完全に解消されていること
  • • 全ての機能が期待通りに動作すること
  • • クラッシュや重大なバグが存在しないこと
  • • Apple Human Interface Guidelinesに準拠していること
  • • 必要な権限設定とプライバシー情報が適切に設定されていること

収益分配例

例：500DL × 150円 = 75,000円の場合

  • ・控除後売上：52,500円
  • ・当社手数料(45%)：23,625円
  • ・開発者収益(55%)：28,875円
