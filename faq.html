❓ 一般的な質問

BucksBox.aiはどのような会社ですか？

+

BucksBox.aiは、2025年に設立されたiOSアプリ開発・審査代行の専門企業です。「誰もが持つアイデアを実際の収益に変える」という理念のもと、技術的な専門知識がない方でもアプリビジネスに参入できる包括的なサービスを提供しています。

主なサービスは以下の3つです：

  • 自社開発アプリの販売
  • アイデアのアプリ化サービス
  • iOS審査代行サービス

サービス利用に年齢制限はありますか？

+

18歳以上の方にサービスをご利用いただけます。未成年者の方は、保護者の同意を得てからご利用ください。

なお、収益分配に関する契約や著作権譲渡契約については、法的な有効性を確保するため、成人されている方のみを対象とさせていただく場合があります。

海外在住でもサービスを利用できますか？

+

基本的に日本在住の方を対象としたサービスとなっております。海外在住の方については、以下の点をご確認ください：

  • 日本語でのコミュニケーションが可能であること
  • 日本の銀行口座をお持ちであること（収益分配用）
  • 日本の税制に関する処理が可能であること

詳細については個別にご相談ください。

💡 アイデアのアプリ化サービス

どのようなアイデアが採用されやすいですか？

+

以下の条件を満たすアイデアが採用されやすくなります：

  • 明確な課題解決：具体的な問題を解決するアイデア
  • 市場性：十分な需要が見込める分野
  • 技術的実現性：iOSアプリとして実装可能
  • 差別化：既存アプリとの明確な違い
  • 収益性：有料販売に適した価値提供

採用されやすいカテゴリ例：

ユーティリティ、ヘルスケア、仕事効率化、ライフスタイル、教育

アイデアが採用されなかった場合、理由を教えてもらえますか？

+

不採用の場合は、以下のような一般的な理由をお伝えいたします：

  • 市場性の課題（需要が限定的）
  • 技術的実現性の問題
  • 類似アプリとの差別化不足
  • 収益性の見込み不足
  • App Store審査通過の可能性

ただし、企業戦略に関わる詳細な分析内容については、開示できない場合があります。

報酬はいつ支払われますか？

+

支払いスケジュール：

  • 対象期間：アプリ公開から30日間の売上
  • 計算：初月売上（Apple手数料控除後）の18%
  • 支払時期：初月終了後、翌月末までに振込
  • 最低支払額：1,000円（未満の場合は次月繰越なし）
重要事項

2ヶ月目以降の売上に対する報酬は一切ありません。初月のみの1回限りの支払いとなります。

開発期間はどのくらいかかりますか？

+

標準的なスケジュール：

  • アイデア審査：1-2週間
  • 開発期間：2-6ヶ月（アプリの複雑性による）
  • Apple審査：1-4週間
  • 公開：審査承認後24時間以内

複雑な機能や高度なデザインが必要な場合は、開発期間が延長される可能性があります。

🛡️ iOS審査代行サービス

審査代行サービスの利用条件は何ですか？

+

必須条件（すべて満たす必要があります）：

  • ビルドエラーゼロでの完全動作保証
  • アプリの著作権を含む全権利の当社への完全譲渡
  • アプリトップ画面への「BucksBox.ai」社名の明確表示
  • 当社指定のロゴを適切なサイズ・位置で表示
  • GitHubプライベートリポジトリでのソースコード提供
重要

これらの要件を1つでも満たせない場合、審査代行をお受けできません。

著作権譲渡は本当に必要ですか？

+

はい、必須条件です。以下の理由により、著作権の完全譲渡が必要です：

  • App Store申請時の法的責任を当社が負担するため
  • 継続的なアプリ運営・アップデートの権限確保
  • 知的財産権に関するトラブル回避
  • Apple規約との整合性確保
永続的な譲渡

一度譲渡された権利は、契約解除等のいかなる理由でも返還されません。この点をご理解いただけない場合は、サービスをご利用いただけません。

Apple審査で却下された場合はどうなりますか？

+

審査却下時の対応：

  • 却下理由の詳細分析と対応策の検討
  • 修正可能な問題については無料で対応
  • 必要に応じてApple審査チームとの調整
  • 再申請の実施（追加費用なし）

ただし、以下の場合は対応が困難です：

  • Apple規約の根本的な違反
  • 技術的に解決不可能な問題
  • アプリのコンセプト自体の変更が必要な場合

事前審査：当社では品質基準を満たすアプリのみ受付し、事前審査により審査通過率を高めています。

収益分配率は変更されることがありますか？

+

はい、変更の可能性があります。

現在の分配率：当社45%、開発者55%（Apple手数料控除後）

変更される場合：

  • 市場環境の大幅な変化
  • App Storeの手数料変更
  • 法令・税制の改正
  • サービス運営コストの著しい変動

変更手続き：30日前までに書面またはメールで通知いたします。

💳 料金・支払い

初期費用や月額費用はかかりますか？

+

いいえ、初期費用・月額費用は一切かかりません。

当社のビジネスモデルは完全成果報酬型です：

  • アイデアのアプリ化：初月売上の18%のみ（2ヶ月目以降は0%）
  • iOS審査代行：継続的に45%の手数料（開発者は55%受取）
  • 自社開発アプリ：App Store価格での販売（当社収益）

売上が発生しない限り、費用は一切発生いたしません。

支払い方法と支払日は？

+

支払い方法：銀行振込（振込手数料は当社負担）

支払いスケジュール：

  • アイデアのアプリ化：初月終了後、翌月末までに振込
  • iOS審査代行：毎月末締め、翌月末払い

最低支払額：

  • アイデアのアプリ化：1,000円
  • iOS審査代行：3,000円

最低支払額に満たない場合の取扱いが異なりますので、詳細は利用規約をご確認ください。

税金の扱いはどうなりますか？

+

重要：税務申告は利用者の責任です

当社の対応：

  • 年間支払総額の報告書を翌年1月末までに送付
  • 必要に応じて支払調書の発行
  • 税務署への法定調書の提出

利用者の責任：

  • 確定申告での収入申告
  • 所得税・住民税の納付
  • 必要に応じて税理士への相談

目安：年間20万円以上の収入がある場合は、確定申告が必要になる可能性があります。詳細は税理士や税務署にご相談ください。

⚖️ 法的事項

著作権譲渡後、自分のアイデアを他で使うことはできますか？

+
重要事項

譲渡したアイデア・アプリと同一または類似のものを他で使用することはできません。

譲渡される権利の範囲：

  • アプリのアイデア・コンセプト
  • 機能・操作方法
  • デザイン・UI/UX
  • 名称・ブランディング
  • 派生・改良版の開発権

これらの権利は永続的に当社に帰属し、利用者は同種のサービス・アプリを開発・販売することができなくなります。

契約を解除したい場合はどうすれば良いですか？

+

解除手続き：

  • 書面またはメールでの解除通知
  • 理由の記載は不要
  • 即日または指定日での解除が可能
解除後の権利関係

契約解除後も以下の権利は当社に帰属し続けます：

  • 既に譲渡されたアイデア・アプリの一切の権利
  • 著作権および知的財産権
  • 成果物・派生物等の権利

当社は解除後も該当アプリの販売・改良を継続できます。

個人情報はどのように保護されていますか？

+

当社では以下の措置により個人情報を保護しています：

  • 技術的措置：SSL/TLS暗号化、ファイアウォール、ウイルス対策
  • 物理的措置：サーバー設備の適切な管理、アクセス制限
  • 組織的措置：個人情報保護管理者の設置、従業者教育
  • 人的措置：秘密保持契約、アクセス権限管理

準拠法規：

  • 個人情報保護法
  • GDPR（EU居住者の情報を扱う場合）
  • CCPA（カリフォルニア州居住者の情報を扱う場合）

詳細はプライバシーポリシーをご確認ください。
